Learning how to learn part 3

一.拖延症

1.如何应对拖延症

  • 专注于学习过程而不是学习结果

  • 排除干扰，防止出现拖延行为

  • 制定任务计划

    2.改变拖延的习惯

  • 习惯的四要素

    • 信号
    • 行为
    • 奖励
    • 信念

  • 改变方式

    • 排除易受干扰的信号
    • 改变接受到信号的拖延行为
    • 给自己奖励，最好与拖延行为得到的满足感一致
    • 自信，相信自己可以改变

二.记忆

1.提高记忆能力

  • 高度视觉化（图像）
  • 空间记忆
  • 重复训练

2.记忆技巧

  • 缩写法
  • 与难忘的事关联起来
  • 记忆宫殿技巧
  • ……
